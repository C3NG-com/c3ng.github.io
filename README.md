# c3ng.com

个人网站 c3ng.com 的 GitHub Pages 静态页面。

## 目录结构

```
c3ng.github.io/
├── assets/              # 静态资源文件
│   ├── css/            # 样式表
│   │   └── style.css   # 主样式文件
│   ├── images/         # 图片资源
│   │   └── c3ng.png    # 网站 logo
│   └── js/             # JavaScript 文件（预留）
├── src/                # 源代码/页面模板
│   └── pages/          # 其他页面（预留）
├── docs/               # 文档资料
│   └── ARCHITECTURE.md # 项目架构说明
├── .gitignore          # Git 忽略文件
├── index.html          # 网站入口页面
└── README.md           # 项目说明
```

## 预览

访问: [https://c3ng.com](https://c3ng.com)

## 技术栈

- 纯 HTML5 + CSS3
- 响应式设计
- CSS 动画效果

## 本地开发

直接在浏览器中打开 `index.html` 即可预览。

## 部署

GitHub Pages 会自动从 `main` 分支部署。
