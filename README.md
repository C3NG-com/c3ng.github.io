# 层构智能科技有限公司

层构智能科技有限公司官方网站。

## 网站结构

```
c3ng.github.io/
├── assets/
│   ├── css/
│   │   └── style.css      # 网站样式
│   ├── images/
│   │   └── c3ng.png       # 公司Logo
│   └── js/                # JavaScript文件（预留）
├── src/
│   └── pages/             # 页面模板（预留）
├── docs/
│   └── ARCHITECTURE.md    # 项目架构文档
├── index.html             # 首页
├── about.html             # 关于我们
├── services.html          # 服务介绍
├── contact.html           # 联系方式
├── .gitignore
└── README.md
```

## 页面说明

- **首页** (`index.html`) - 公司介绍、核心业务展示
- **关于** (`about.html`) - 公司简介、理念、技术领域
- **服务** (`services.html`) - 核心服务、服务流程、行业应用
- **联系** (`contact.html`) - 联系方式、业务咨询

## 设计风格

- 白底黑字终端风格
- 等宽字体 (Courier)
- 线条边框装饰
- 无阴影渐变，简洁硬朗

## 本地预览

```bash
python -m http.server 8080
```

访问: http://localhost:8080

GitHub Pages 自动从 `main` 分支部署。
