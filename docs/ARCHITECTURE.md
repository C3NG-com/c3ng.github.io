# 项目架构文档

## 目录说明

### `/assets/`
存放所有静态资源文件

- **`/assets/css/`** - 样式表文件
  - 按页面或组件组织 CSS
  - 使用 `style.css` 作为主样式入口

- **`/assets/images/`** - 图片资源
  - logo、图标、背景图等
  - 建议使用子目录按用途分类

- **`/assets/js/`** - JavaScript 文件
  - 交互逻辑
  - 工具函数

### `/src/`
源代码目录

- **`/src/pages/`** - 其他页面模板
  - 关于页面、博客页面等

### `/docs/`
文档资料

- 项目说明、API 文档等

## 开发规范

1. **文件命名**: 使用小写字母和连字符，如 `main-style.css`
2. **图片格式**: 优先使用 PNG（透明）、JPG（照片）、SVG（图标）
3. **代码风格**: 保持缩进一致（2个空格）
